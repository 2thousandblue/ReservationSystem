# 航空订票系统

### 系统的总体设计
本系统采用的CS结构，因此，此系统可分成页面的设计，包括管理员页面和用户页面；业务层的设计，主要是一些关于处理页面发过来业务请求的处理方法。数据层设计，主要是对业务层发来的数据请求的响应。
### 系统功能
**用户端**
1. 用户注册
2. 修改用户信息
3. 查询航班
4. 预订机票
5. 订单改签
6. 退票
